To create a sleet button.

Let's start

Right-click on the desktop → Choose New → Shortcut.

In the location box, paste this:"rundll32.exe powrprof.dll,SetSuspendState 0,1,0"
Click Next, name it something like "Sleep", then click Finish.
(Optional) Right-click the new shortcut → Properties → Change Icon → pick a sleep-related icon.
If it says no icons found, paste:%SystemRoot%\System32\shell32.dll
Choose an icon → Click OK > Apply
Done
